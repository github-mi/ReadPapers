# 公式说明

使用[readme2tex](https://github.com/leegao/readme2tex) 将Latex公式转为文本,\*.tex.md是纯文本格式，转化为对应的\*.md(公式转为svg图片)。

```python
python -m readme2tex --output README.tex.md README.md --nocdn
```

> 在使用readme2tex时将needs_inversion相关代码注释掉了。
